## Getting started

Add this to your SBT project:

```
resolvers += "dl-john-ky-releases" at "http://dl.john-ky.io/maven/releases"

libraryDependencies += "org.pico" %%  "pico-event" % "2.0.2"
```

Then read the [tutorial](pico-event/src/main/tut/tutorial.md).
